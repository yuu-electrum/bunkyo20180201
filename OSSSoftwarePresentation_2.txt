- MySQL

1) https://www.mysql.com/jp/　（Oracle）

2) MySQLは、いわゆるRDBMS(Relational Database Management System、関係データベース管理システム)のひとつである。世界でもっとも使われているRDBMSだ。

3) 最初期のMySQLは1994年に開発され、公開は1995年である。2001年に現在のMySQLに近いものとなっている。ストレートエンジンと呼ばれるMySQLのトランザクション処理やインデックス管理などについて固有の処理を持たせる機構がある。現在のシェアトップであるOracleはそのうちの一つであるInnoDBを開発していたInnobase Oyを買収している。

    3) http://gihyo.jp/dev/serial/01/MySQL-tuning-scale/0001
    
4) 利用形態によって異なるライセンス形式となる。GNU GPLによって再配布「しない」場合、商用のライセンスが必要となる。

Apache
1)URL: https://httpd.apache.org/
2)機能概要:HTTPサーバー
 特徴:モジュールを追加することで拡張可能
3)沿革:初版    1995年
 初期の開発者(立ち上げ):Brandon Long,Beth Frank
　開発元:Apache Software Foundation
4)
 利用条件:
    利用する際、ライセンス契約のために電子署名が必要。
　配布条件:
    成果物または派生成果物の他の受領者に本ライセンスのコピーも渡すこと。
    変更を加えたファイルについては、あなたが変更したということがよくわかるような告知を入れること。
    ソース形式の派生成果物を頒布する場合は、ソース形式の成果物に含まれている著作権、特許、商標、および帰属についての告知を、派生成果物のどこにも関係しないものは除いて、すべて派生成果物に入れること。
    成果物の一部として「NOTICE」に相当するテキストファイルが含まれている場合は、そうしたNOTICEファイルに含まれている帰属告知のコピーを、派生成果物のどこにも関係しないものは除いて、頒布する派生成果物に入れること。
    その際、次のうちの少なくとも1箇所に挿入すること。
    (i) 派生成果物の一部として頒布するNOTICEテキストファイル、
    (ii) ソース形式またはドキュメント（派生成果物と共にドキュメントを頒布する場合）、
    (iii) 派生成果物によって生成される表示（こうした第三者告知を盛り込むことが標準的なやり方になっている場合）。
    NOTICEファイルの内容はあくまで情報伝達用であって、本ライセンスを修正するものであってはなりません。
    あなたは頒布する派生成果物に自分の帰属告知を（成果物からのNOTICEテキストに並べて、またはその付録として）追加できますが、これはそうした追加の帰属告知が本ライセンスの修正と解釈されるおそれがない場合に限られます。

- PHP: Hypertext Preprocessor

1) http://php.net/

2) 汎用スクリプト言語のひとつであり、実行されるとファイルの先頭から逐次翻訳（コンパイル）および実行がなされ、いわゆるインタプリタ形式のスクリプト言語である。HTMLとの親和性が非常に高く、広くweb開発で使われている。

3) Rasmus Lerdorfがweb上で公開している、自身の経歴書へのアクセス記録を調べるためのスクリプトの集合である「Personal Home Page Tools」が発端となっている。Rasmusはそれらのソースコードを1995年に公開し、ユーザーが自身でそれらに含まれるバグを改修することや機能を追加することを推奨した。同年9年、PHPの文字が消え、それらはForms Interpreterの名称を与えられるが、翌月1995年10月にまたPHPと名乗りだし、「Personal Home Page Construction Kit」になった。現在のPHPに近い状態となったのはPHP 3である。


    3) http://php.net/manual/ja/history.php.php
    
4) PHP Licenseによって配布される。ただし、一部のファイルに関しては他のライセンスによる。BSD Licenseをベースとしている。

5) http://windows.php.net/download/

- GNU Image Manipulation Program

1) https://www.gimp.org/

2) GNU Image Manipulation Program（GIMP）は、クラスプラットフォームに対応している画像エディタである。無料でありながら、非常に強力な編集機能を備え、その対応形式も多岐に渡る。また、スクリプトによってその機能を拡張することができる。

3) Peter MattisとSpencer Kimball。
「Version 0.54 は 1996年2月に公開され、 最初の真に実用的でフリーな画像編集プログラムとして、 大きな衝撃をもって迎えられました。 かの大壮な商用の画像編集プログラムに匹敵しうる最初のフリーなプログラムでもありました。

Version 0.54 はベータ版ですが、 日々の使用に耐えうる安定した版でした。 しかしながら 0.54 が使うツールキット (サイドバーやメニューやダイアログボックスなどの部品) が Motif® だったので、 商用ツールキットを使用していることが大きな障害のひとつとなりました。 この仕様では GIMP を高速に動かそうと思ったら動的リンクさせるために Motif® を購入しなくてはならず、 Linux™ のようなシステムにとっては大きな足かせになります。 開発には Linux™ を使う学生も携わっており、 Motif® を買う余裕のない者が大勢いました。」

4) GNU GPL。

5) https://www.gimp.org/source/
